# web-dev setup

Tasks

## install dependencies

```sh
npm install
```

## run in development mode

```sh
npm start
```

## build for production

```sh
npm run build
```

## test

```sh
npm test
```

### Features

- ES2015+
- Babel
- React
- Allows live tweaking of webpack config and restarts dev environment when config gets changed
